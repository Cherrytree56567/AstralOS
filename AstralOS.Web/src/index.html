Astral OS

A simple AMD 64 OS that aims to be as good as Linux. It currently supports ACPI, APIC, Basic Console, Drivers, ELF, GDT, IDT, InitRD, IOAPIC, Paging, Heap, PCI/PCIe, Timing and AHCI Drives. Here is a QEMU VM Powered By ktock/qemu-wasm to showcase the OS. It will take a while to load.
